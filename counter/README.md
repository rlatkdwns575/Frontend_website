# Counter App

간단한 웹 기반 카운터 애플리케이션입니다.

버튼을 눌러 값을 1 또는 5씩 증가/감소시킬 수 있으며, 리셋 기능과 최대/최소 범위 제한을 포함합니다. 또한 현재 값이 양수인지 음수인지에 따라 숫자 색상이 변경되도록 구성했습니다.

## 주요 기능

- `+1`, `+5`, `-1`, `-5` 버튼을 통한 카운트 조작
- `리셋` 버튼을 통한 값 초기화
- 카운트 값에 따른 텍스트 색상 변경
  - 양수: 초록색
  - 음수: 빨간색
  - 0: 기본 색상
- 허용 범위 제한
  - 최대값: `50`
  - 최소값: `-50`
- 범위를 초과하거나 리셋할 경우 알림창 표시

## 프로젝트 구조

```bash
.
├── counter.html
├── style.css
└── app.js
```

### 파일 설명

- `counter.html`  
  카운터 UI의 기본 구조를 정의한 HTML 파일입니다.

- `style.css`  
  화면 중앙 정렬, 버튼 배치, 버튼 색상, 카운터 숫자 스타일 등을 담당합니다.

- `app.js`  
  버튼 클릭 이벤트 처리, 숫자 갱신, 최대/최소값 제한, 색상 변경 로직을 담당합니다.

## 실행 방법

1. 세 파일(`counter.html`, `style.css`, `app.js`)을 같은 폴더에 저장합니다.
2. `counter.html` 파일을 브라우저에서 엽니다.
3. 버튼을 클릭하여 카운터를 조작합니다.

## 동작 방식

- 초기값은 `0`입니다.
- 증가/감소 버튼을 클릭하면 값이 변경되고 화면에 즉시 반영됩니다.
- 값이 `50`을 초과하면 `50`으로 고정되며 경고창이 표시됩니다.
- 값이 `-50`보다 작아지면 `-50`으로 고정되며 경고창이 표시됩니다.
- `리셋` 버튼을 누르면 값이 `0`으로 돌아가고 초기화 알림이 표시됩니다.
